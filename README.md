# University System

This project is a Java application that helps to manage tasks from an university  
The system allows users to manage teachers, students, and courses through a console menu

# Features

The application allows users to:

- View all teachers and their salary information
- View all courses and their details
- Create new students and assign them to a course
- Create new courses and assign students and teachers
- Search for all courses in which a student is enrolled

# OOP Concepts Implemented

This project implements the concepts:

- Encapsulation
- Inheritance
- Polymorphism
- Constructors
- Static attributes and methods
- Layered architecture

# Project Structure

The project is organized into layers using packages:

**main**
- Contains the main runnable file of the app

**ui**
- This is where the menu is implemented

**service**
- This is the business logic of the app

**model**
- Contains the classes:
  - Student
  - Teacher
  - FullTimeTeacher
  - PartTimeTeacher
  - Course
  - University
